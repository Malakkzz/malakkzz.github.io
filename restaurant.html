Restaurant Website

Home Page
Includes an admin panel, where an superadmin and admin are present. The superadmin is the only one who can add other admins.>

Login Page
Users saved in the database, where if one or both of the fields are entered wrong, it won't allow the user access to the website.

Signup Page
The user must fill all of the fields so that he can have an account on the website. He can click on the "already member" button if he has an account before.

Reservation Page
There are specific conditions for the user to book a table. He can't register more than 8 guests, and the maximum number of guests allowed are 25. In addition to that, he can leave a suggestion for the restaurant below!

Offers Section
The admin is the only one that has access to adding offers, and the user can benefit from it until it expires.

Menu Page Showing all the categories in the menu and for sure when the button is clicked, the items in each category will be shown.

Feedback Page
Bunch of feedbacks left by users that tried the restaurant with the rating followed.

Currency and Add Item Features
Only the admin can add items and change the currency in the website>

Categories Section
It is shown as a section in the home page.
